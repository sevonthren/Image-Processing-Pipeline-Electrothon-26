# Image-Processing-Pipeline-Electrothon-26
A simple MATLAB script along with the images (both unfiltered and cleaned) as my submission for the Signal Processing module for the Electrothon held by the Electronics Club IITG. The pdf file is a detailed report on the MATLAB code as required per the problem statement. 
Note that the 'schem_output' folder contains two sets of images: processed and diagnostic. The processed images are the final images after image processing and the diagnostic images contain the diagnostics as described in the report.
